# 内置中文 Skills

macOS 原生应用随包提供 5 项由本项目编写的中文基础能力，并在首次启动时自动安装到 `~/.codex/skills`。用户不需要自行搜索或配置，就可以使用任务托管、产品判断、UI 验收、完成前检查和 Skills 中文向导。

源文件位于 `macos/Sources/CodexSkillHelper/Resources/StarterSkills`，与本仓库一起按 MIT License 发布。

项目不会直接复制作者电脑里的全部 Skills。系统 Skills 已由 Codex 提供；第三方 Skills 只有在许可证允许、来源清楚并通过安全检查后，才适合进入可分发能力包。
